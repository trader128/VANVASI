## Description

VANVASI is monk mode for your iPhone.

When VANVASI Lock is on, only Phone and Messages stay open. Everything else is shielded by Apple's Screen Time API — the same technology trusted by top focus apps.

Before you unlock, take a breath. VANVASI pauses you, then grants timed access if you still need it. When the timer ends, monk mode returns automatically.

FEATURES
• Hard lock — system-level shields, not easy to ignore
• Intentional unlock — breathing pause before access
• Focus score & streak — track your progress
• Scheduled lock — auto-enable during work or sleep hours
• PIN protection — extra friction to disable lock
• Shortcuts — "Enable VANVASI Lock" from Siri and automations
• Widget — lock status at a glance
• Private — no account, no cloud, all on your device

Built for traders, students, and anyone whose focus is their edge.
